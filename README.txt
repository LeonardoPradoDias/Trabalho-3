O objetivo do trabalho é implementar um programa na linguagem C a fim de realizar operações simples em uma matriz esparsa de tamanho definido pelo usuário. 
O programa permite a criação da matriz, inserção de valores nas posições "i" (linha) e "j" (coluna), consulta do valor de uma posição, consulta da soma dos valores de uma determinada linha ou coluna e exclusão da matriz. 
O programa também possui diversas verificações para evitar que o usuário insira dados inválidos como letras, por exemplo. 
A estrutura do código é baseada nas ferramentas de registro de dados.
